# 感想

  • 单元测试对工程师的要求门槛较高，要有很好的代码能力
  • 前端项目不一定适合 TDD 单元测试，更适合的应该是 BDD
  • 要推动 TDD，一定需要改革整个研发的流程，这个往往是十分困难的，也正是这个原因，实际开展tdd的项目也不是很多。
